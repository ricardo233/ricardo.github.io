Analysis:

Firstly, we plot all cargo number together with ULD number on one chart and this is what we get:

￼

As indicated in the plot, it shows a linear relationship between total number of cargos and number of ULDs. Considering each ULD should carry integer number of Cargos and the more ULDs come, the more cargos we should have ( zero cargo if no ULD comes). It's reasonable to suppose the total number of cargo (Y) and number of ULDs (x) follows a relationship of:
\[Y = \beta x\]
where \(\beta\) represent the mean value of number of cargos per ULD.
Furthermore, if we apply linear regression analysis to the model,
￼

we can get the result of
\[\hat{\beta}=6.9998\quad and \quad R^2=0.9672\]
The confidence interval of \(\beta\) is (6.8878, 7.1118) at significant level of 0.05. This shows the mean value of number of cargos per ULD is equal to 6.9998.

Moreover, if we look at the plot of the cargo/ULD ratio on one histogram, we can observe that the ratio are restricted in the interval [6,8]. And, the sample is roughly uniformly distributed in the interval with a sample mean \(\bar X = 6.96694\) and sample variance \(S^2 = 0.4293\)

￼

From this, it's reasonable to model the distribution of number of cargos per ULD with a discrete uniform distribution (i.e \(X\)~\(DU(6,8)\)), which has a population mean \(\mu = 7\) and population variance \(\sigma^2 = 2/3\)

\(X\)~\(DU(6,8)\) Mean Variance
Population 7 0.6667
Sample 0.69998 0.4293

Conclusion:
Discrete Uniform distribution from 6 to 8 is used to model the number of cargos per ULD from different origin. So the total number of cargos is calculated by \(ULDs * DU(6,8)\).
